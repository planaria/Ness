【新規導入手順】

1. VRCSDK3, UdonSharp をインポートし、ワールドをセットアップします。

2. Ness.unitypackage をインポートします。

3. Unity のメニューから "VRChat SDK" > "Utilities" > "Re-compile All Program Sources" を実行します。

4. Assets/Ness/Prefabs/PanelPool.prefab をシーンのルート階層に配置します。見えないところに置いたほうがいいです。

5. Assets/Ness/Prefabs/EditorBoard.prefab を好きなところに配置します。

6. Assets/Ness/Prefabs/Eraser.prefab を好きなところに配置します。

7. Assets/Ness/Prefabs/LaserPointer.prefab を好きなところに配置します。

8. Assets/Ness/Prefabs/PanelBuilder.prefab を好きなところに配置します。

9. Assets/Ness/Prefabs/TwinPanelBuilder.prefab を好きなところに配置します。

【アップデートする場合の作業】

1. Ness.unitypackage をインポートします。

2. Unity のメニューから "VRChat SDK" > "Utilities" > "Re-compile All Program Sources" を実行します。

3. 今までシーンに配置した prefab それぞれについて、選択した状態で Inspector の右上の Overrides をクリックし、
　 下に表示された "Revert All" をクリックします。"Revert All" が表示されていない場合は何もしなくて大丈夫です。

4. 追加する prefab を好きなところに配置します。
